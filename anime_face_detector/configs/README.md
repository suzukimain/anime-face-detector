# Config Files (Legacy)

**Note**: These config files are for the old OpenMMLab-based implementation (v0.0.9 and earlier).

The new version (v0.1.0+) uses **Ultralytics YOLO** and does not require these config files.

## Migration

- Old: `mmdet` + `mmpose` + config files
- New: `ultralytics` YOLO (no config needed, models auto-download)

If you need the old behavior, please use version 0.0.9 or earlier.
